About Me

"Halo! Saya adalah seorang siswa SMK Jurusan Pengembangan Perangkat Lunak yang memiliki ketertarikan mendalam pada dunia rekayasa perangkat lunak. Bagi saya, koding bukan sekadar menulis baris instruksi agar aplikasi berjalan, melainkan sebuah seni memecahkan masalah lewat logika yang terstruktur.

Saat ini, saya fokus mendalami Web Development, mulai dari menyusun fondasi dengan HTML5, merancang visual modern yang responsif menggunakan CSS3, hingga membangun interaktivitas lewat JavaScript. Saya sangat menyukai arsitektur kode yang bersih (clean code) dan tampilan futuristik yang memanjakan pengguna.

Tidak berhenti di situ, saya juga aktif mempelajari bahasa pemrograman lain seperti Python dan Java untuk memperkuat pemahaman saya tentang algoritma dan konsep Object-Oriented Programming (OOP). Perjalanan ini adalah langkah awal saya untuk mencapai cita-cita besar: menjadi seorang Software Engineer profesional yang mampu membangun solusi digital yang efisien dan berdampak nyata."
